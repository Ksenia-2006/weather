# Weather с Яндекс.Картами
Интерактивное веб-приложение для отображения текущей погоды по координатам с интеграцией Яндекс.Карт.

## Особенности

### Основной функционал
- Добавление нескольких виджетов погоды для разных координат
- Валидация координат с проверкой корректности ввода
- Интерактивная карта с маркерами для каждого виджета
- Кэширование данных на 1 час для оптимизации

### Отображение погоды
- Текущая температура (в градусах по Цельсию)
- Описание погоды (ясно, облачно, дождь и т.д.)
- Дополнительная информация:
  - Влажность (%)
  - Скорость ветра (м/с)
- Иконки погоды, соответствующие текущим погодным условиям

### Функции карты
- Интеграция с Яндекс.Картами API
- Цветные маркеры для каждого виджета
- Всплывающие окна с подробной информацией
- Навигация между виджетами и картой
- Автоматическое центрирование карты (Екатеренбург по умолчанию)
